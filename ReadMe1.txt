6 files are included.


1. distances.txt		Distance observations in a tab limited format
	File format:
	From	To	Observation (in meter)	Accuracy (in meter)


2. angles.txt: 		Horizontal angle observations in a tab limited format
	File format:
	From 	At	To	Observation (in Degrees)		Accuracy (in Seconds)


3. coords.txt		Control Point coordinates (fixed coordinates) in a tab limited format
	File format:
	Point number		X (in meters)		Y (in meters)


4.  Dist	.m		A function to calculate Distance using approximate coordinates in meters


5. Ang.m: 		A function to calculate horizontal angle in radians using approximate coordinates


6. Chi.txt:			Chi square distribution table
